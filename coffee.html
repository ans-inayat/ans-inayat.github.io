Coffee Cove Shop - Bug Bounty & Vulnerability Testing

Description: Coffee Cove Shop is a custom-built coffee shop web application that underwent thorough bug bounty testing and vulnerability assessments. During the testing, critical vulnerabilities such as Cross-Site Scripting (XSS) and SQL Injection were identified and mitigated. The app utilizes a secure backend and ensures users' privacy through robust security practices.

Technologies Used: PHP, MySQL, Laravel, Penetration Testing

Project Duration: 4 Months

View on GitHub Live Demo

Key Features

  • Bug Bounty Program: Engaged with security researchers to identify vulnerabilities.
  • Vulnerability Testing: Focused on XSS and SQL Injection vulnerabilities, with successful mitigation.
  • Security Best Practices: Implemented secure coding practices and input validation to enhance security.
